# Netflix TV Shows & Movies – Exploratory Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset.
The objective is to understand content distribution, trends, duration patterns, ratings, and country-wise production
using Python data analysis libraries.

This project is completed as **Task 2 (EDA)** for the **CodeAlpha Data Analytics Internship**.

---

## 📊 Dataset
- Source: Kaggle – Netflix Movies and TV Shows Dataset
- File name: `netflix_titles.csv`
- Records include movies and TV shows available on Netflix with details such as:
  - Type
  - Country
  - Release Year
  - Rating
  - Duration
  - Genre

---

## 🛠️ Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🔍 Key Analysis Performed
- Movies vs TV Shows count analysis
- Year-wise content growth on Netflix
- Country-wise content distribution
- Rating distribution analysis
- Genre popularity analysis
- TV Shows vs Movies duration analysis

---

## 📈 Key Insights
- Netflix content has grown rapidly after 2015.
- Movies are more numerous than TV shows on the platform.
- Most movies are between 80–120 minutes long.
- Most TV shows have only 1–2 seasons.
- The USA is the top content-producing country for Netflix.
- TV-MA is the most common rating.

This exploratory data analysis provides a clear understanding of Netflix’s content strategy.

and content distribution trends on Netflix.
